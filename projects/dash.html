# Project Name : Dash

# YEAR : 2020

# Tech Stack

VueJS, Firestore, Cloud Functions, Google Maps, Vuetify, Android

# Merchant App ( Web )


# Objective

To create a logistic delivery service application that would allowed user easily create the logictis delivery order.

# Target User

The application is aimed for small and medium businesses owner to be able to provide delivery service to their consumer without having their own logisctic delivery service in-house.

# Application Features

  • Single Page Order Form
  • Google Maps Views showing the location of the pickup and dropoff address
  • Car & Bike Delivery Options
  • Schedule Pickup Time
  • Display of Total Delivery Trip Cost

# Back Office Portal ( Web )


# Objective

To manage all the operations of logistic delivery orders that receive from the web application, and manages all the business users and delivery drivers accounts.

# Target User

The operation team in Dash

# Application Features

  • Business Account Creation
  • Business Account Credits Top Up
  • Delivery Drivers Account Verifications
  • Orders Delivery Progress & Overview
  • Sales & Data Dashboard

# Driver App ( Android )


# Objective

To notify nearest delivery drivers on the new incoming delivery order.

# Target User

Delivery Driver

# Application Features

  • Driver Accounts Registration
  • New Incoming Orders Notification
  • Order Status & Updates
