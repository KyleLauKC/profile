# Project Name : Vollo

# YEAR : 2021- 2022

# Tech Stack

Nuxtjs, Firestore, Firebase Hosting, Vuetify, i18n

# User App ( Web )


# Objective

To allow user to create food order to restaurant in their own finger tips

# Target User

Everyday Consumers

# Application Features

  • View Store Menu
  • Create Delivery/ Takeout Food Order
  • Order Status Update

# Merchant Web Application ( Web )


# Objective

To allow businesses to have their own virtual menu app that provides delivery and takeout service without relying on third party delivery platform.

# Target User

Food and Beverages Business Merchant

# Application Features

  • Stores information & details
  • Store QR Link
  • Virtual Menu
  • Manage & Update Food Delivery/Takeout Ticket
  • Daily and Monthly Orders data View
  • Daily and Monthly Transactions data View
  • Mutliingual Options Supported (English, Chinese & Malay)
